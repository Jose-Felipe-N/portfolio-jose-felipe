# 👨‍💻 Portfólio Pessoal | José Felipe

Bem-vindo ao repositório do meu portfólio web! Este projeto foi desenvolvido inteiramente em **Python** utilizando o framework **Streamlit** para apresentar de forma interativa minha trajetória, habilidades e os principais projetos que venho desenvolvendo.

## 🎯 Sobre Mim
Sou estudante de Ciência da Computação na Universidade Federal do ABC (UFABC). Tenho um forte interesse na intersecção entre matemática, física e tecnologia, buscando sempre aplicar o pensamento analítico para otimizar processos e criar sistemas eficientes. 

## 🚀 Estrutura do Site
O portfólio é dividido em três seções principais:
* **Sobre Mim:** Detalhes sobre minha formação profissional e acadêmica.
* **Projetos:** Uma vitrine com links e descrições das minhas principais aplicações (como o Sistema de Triagem Financeira).
* **Contato:** Links rápidos para o meu GitHub, LinkedIn e e-mail.

## 🛠️ Tecnologias Utilizadas
